# Inside Airbnb — Data Visualization Dashboard

An interactive data visualization dashboard built for the **HCMUS Data Visualization** course. It explores the [Inside Airbnb](http://insideairbnb.com/) dataset through six analytical domain tasks, rendered as a Tableau-style single-page application.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | [React 19](https://react.dev/) + [TypeScript 6](https://www.typescriptlang.org/) |
| Build tool | [Vite 8](https://vitejs.dev/) |
| Visualization | [D3.js v7](https://d3js.org/) |
| Styling | Vanilla CSS (custom design system) |
| Data preprocessing | Python 3 + pandas |

## Data Files

All derived CSV files used by the dashboard are already included in `public/data/derived/` — **no data setup is needed to run the app**.

The GeoJSON boundary file is also included:

```
public/data/neighbourhoods.geojson
```

> If you want to adjust the data or re-run the preprocessing pipeline, see the [Optional — Regenerate derived data](#optional--regenerate-derived-data) section below. The three raw source CSVs (`listings_cleaned.csv`, `calendar_cleaned.csv`, `reviews_cleaned.csv`) are not committed due to their size (~1 GB total) and must be obtained separately from the Inside Airbnb data export.

## How to Run

### Prerequisites

- **Node.js** ≥ 18

---

### Step 1 — Install dependencies

```bash
npm install
```

---

### Step 2 — Start the dev server

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

---

### Optional — Regenerate derived data

Only needed if you want to adjust the source data or re-run the preprocessing pipeline. Requires Python ≥ 3.9 and `pandas`.

First, place the three raw source CSVs in `public/data/`:

| File | Description | Approx. size |
|---|---|---|
| `public/data/listings_cleaned.csv` | One row per listing — price, ratings, host flags, coordinates | ~9.3 MB |
| `public/data/calendar_cleaned.csv` | Daily availability & price per listing for 365 days | ~663 MB |
| `public/data/reviews_cleaned.csv` | One row per review — listing ID, date, room type | ~371 MB |

Then run the pipeline:

```bash
# Full pipeline (Tasks 1–6) — takes 5–15 min due to calendar_cleaned.csv size
python scripts/preprocess_derived_data.py

# Or regenerate a single task
python scripts/regen_task3_only.py
python scripts/regen_task4_only.py
```

---

### Optional — Production build

```bash
npm run build
npm run preview
```

## Domain Tasks

| # | Title | Chart idiom |
|---|---|---|
| 1 | Price–Rating Correlation by Neighbourhood & Room Type | Diverging bar chart |
| 2 | Seasonal Review Demand & Listing Promotions | Stacked bar chart + drilldown |
| 3 | Daily Availability & Price Calendar | Split-cell calendar heatmap |
| 4 | Minimum-Night Policy & Price Rigidity vs Vacancy | Box plot |
| 5 | Top-tier Superhost Spatial Density | Spatial density heatmap |
| 6 | Host Benchmark Profile | Radar / profile chart |

Full bilingual (Vietnamese / English) task documentation lives in `.KB/DomainTasks/`.

## Scripts Reference

| Script | Purpose |
|---|---|
| `scripts/preprocess_derived_data.py` | Full pipeline — generates all derived CSVs for Tasks 1–6 |
| `scripts/regen_task3_only.py` | Regenerate Task 3 daily summary & intervention candidates only |
| `scripts/regen_task4_only.py` | Regenerate Task 4 box-plot & support-candidate files only |
| `scripts/add_listing_name_to_cleaned.py` | Backfill `name` column into `listings_cleaned.csv` |
