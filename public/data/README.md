# Data Folder

Put your cleaned CSV dataset here as:

```text
public/data/chart-data.csv
```

The starter visualization expects at least:

```csv
category,value,group
Jan,1200,Semester 1
Feb,1900,Semester 1
Mar,1500,Semester 1
```

Accepted category column names:

```text
category, label, name, month
```

Accepted numeric value column names:

```text
value, count, total, visitors
```

Optional color/group column names:

```text
group, type, region
```
